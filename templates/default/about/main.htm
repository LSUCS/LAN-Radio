About LAN Radio

What?

LAN Radio is a democratic music player for LAN parties that allows everyone a fair say in what music is played. It supports music from sources such as Spotify and YouTube.

Why?

Originally at LANs we used Blicko, however this was often unstable, and forced annoying social media interractions (such as forcing you to post on your Facebook page when you added or voted for a song).

How?

The web frontend of the system runs on a php framework written by myself. It's designed for a much larger site and is probably overkill for this use. Here is the GitHub link for anyone interested.
The backend of the system uses Mopidy music server, with some additional plugins written for YouTube, and to use a dynamic playlist. It utilizes GStreamer.
